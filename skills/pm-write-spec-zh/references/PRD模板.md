# PRD 模板

## 1. 需求摘要

- 一句话描述本次改动

## 2. 背景与问题

- 背景
- 当前现状
- 当前问题

## 3. 目标与非目标

### 目标

- 本次明确要达成的结果

### 非目标

- 本次明确不做的内容

## 4. 用户与场景

- 目标角色
- 入口
- 核心使用场景

## 5. 方案范围

- 涉及页面 / 模块 / 流程
- 不涉及范围

## 6. 方案说明

- 核心流程
- 页面或模块变化
- 主动作 / 次动作

## 7. 规则与边界

- 状态分支
- 权限分支
- 异常分支

## 8. 联动影响

- 审核
- 消息
- 导出
- 搜索
- 埋点
- 后台

## 9. 验收标准

- 主流程通过
- 关键边界通过
- 旧逻辑不被误伤

## 10. 待确认问题

- 仍需决策的事项
